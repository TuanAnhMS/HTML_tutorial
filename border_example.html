HTML & CSS Borders Guide

1. The Shorthand Property

The most common way to add a border. Syntax: border: width style color;

I have a border: 2px solid black;

2. Individual Properties

You can specify width, style, and color separately.

border-width: 5px;
border-style: dashed;
border-color: #e74c3c;
