HTML Color Examples

1. Color Names (Named Colors)

Red Blue Green Yellow Orange Purple
Pink Cyan Magenta Brown Gray Teal
Navy Maroon Olive Coral Indigo Khaki

2. Hexadecimal Colors (HEX)

Format: #RRGGBB

#FF0000 #00FF00 #0000FF #FFFF00 #FF6347 #4B0082
#FF1493 #228B22 #87CEEB #FFD700 #20B2AA #EE82EE

3. RGB Colors

Format: rgb(red, green, blue)

rgb(255, 0, 0) rgb(0, 255, 0) rgb(0, 0, 255) rgb(255, 165, 0) rgb(128, 0, 128) rgb(255, 192, 203)
rgb(64, 224, 208) rgb(184, 134, 11) rgb(70, 130, 180) rgb(255, 20, 147) rgb(0, 100, 0) rgb(192, 192, 192)

4. RGBA Colors (RGB + Alpha/Transparency)

Format: rgba(red, green, blue, alpha) - alpha: 0.0 to 1.0

rgba(255, 0, 0, 1.0) rgba(255, 0, 0, 0.8) rgba(255, 0, 0, 0.5) rgba(0, 0, 255, 0.3) rgba(0, 255, 0, 0.6) rgba(255, 165, 0, 0.4)

5. HSL Colors (Hue, Saturation, Lightness)

Format: hsl(hue, saturation%, lightness%)

hsl(0, 100%, 50%) hsl(120, 100%, 50%) hsl(240, 100%, 50%) hsl(60, 100%, 50%) hsl(300, 100%, 50%) hsl(180, 100%, 50%)
hsl(0, 100%, 25%) hsl(0, 100%, 75%) hsl(0, 50%, 50%) hsl(120, 50%, 50%) hsl(240, 50%, 50%) hsl(30, 100%, 50%)

6. HSLA Colors (HSL + Alpha/Transparency)

Format: hsla(hue, saturation%, lightness%, alpha)

hsla(0, 100%, 50%, 1.0) hsla(0, 100%, 50%, 0.8) hsla(0, 100%, 50%, 0.5) hsla(120, 100%, 50%, 0.6) hsla(240, 100%, 50%, 0.3) hsla(60, 100%, 50%, 0.4)

7. Hexadecimal Colors with Alpha (HEXA)

Format: #RRGGBBAA

#FF0000FF #FF0000CC #FF000080 #0000FF4D #00FF0099 #FFFF0066
