Method 2: Using PNG Image

Modern browsers support PNG format. Recommended sizes: 16x16px, 32x32px, or 64x64px

Method 3: Using SVG

SVG files are scalable and lightweight. Best for modern browsers.

Method 4: Apple Touch Icon (Mobile)

Recommended size: 180x180px. Used when users add your site to iPhone home screen.
