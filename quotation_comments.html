Quotations, Comments, and Text Formatting (basic HTML)

1. Block quotations

Use <blockquote> for longer, block-level quotations. Add a citation with <cite> if desired.

"The only limit to our realization of tomorrow is our doubts of today."

— Franklin D. Roosevelt

2. Inline quotations

Use <q> for short, inline quotations:

She said I will be there soon and left.

3. Quotations with attribution (common patterns)

Often you show the quote then an attribution on the next line:

Success is not final; failure is not fatal: It is the courage to continue that counts.

— Winston Churchill

4. Quoting other sources (using <cite>)

Use <c ite> for the work or source name, not for a person’s name alone:

From The Elements of Style — Omit needless words.

5. HTML comments

Comments are invisible in the browser and written like this:

Example above is a comment and not rendered.

6. Text formatting tags

  • Bold text using <b> — purely presentational
  • Strong importance using <strong> — semantic emphasis
  • Italic text using <i> — presentational
  • Emphasis using <em> — semantic emphasis (may be read by screen readers)
  • Underlined text using <u>
  • Highlighted text using <mark>
  • Fine print using <small>
  • Deleted text using <del>
  • Inserted text using <ins>
  • Subscript: H2O using <sub>
  • Superscript: x2 using <sup>
  • Code: <code>var x = 1;</code>
  • Sample output: OK
  • Variable: n (used for variables)
  • Keyboard input: Ctrl + C

7. Combined examples

Bold, emphasized, and underlined in one sentence.

Quote with formatting: Be yourself; everyone else is already taken. — Oscar Wilde

8. Notes and best practices

  • Prefer <strong> and <em> for semantic meaning.
  • Use <b> and <i> when you only want visual styling (older HTML behavior).
  • HTML comments () are removed from the rendered page but visible in page source; do not put secrets there.
  • Use <blockquote> for long quotations and <q> for short inline quotes.
